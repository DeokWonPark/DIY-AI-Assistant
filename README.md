### Purpose of the project

- Vue.js를 사용한 프론트엔드 개발을 통해서 Vue.js 학습
- Socket.io를 활용한 서버와 클라이언트의 원활한 실시간 통신
- Dialogflow를 활용하여 자연스럽고, 원활한 대화가 이루어지는 챗봇 기능
